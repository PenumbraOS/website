# PenumbraOS Website

The Hugo base of the PenumbraOS website, hosted at https://penumbraos.com.

Besides the markdown documentation provided here, this repo contains the tooling to pull in other repos that are combined into the final site:

- https://github.com/PenumbraOS/center - The PenumbraOS dashboard and installer webapp
- https://github.com/PenumbraOS/humane-press-kit - Humane's official press kit
- https://github.com/PenumbraOS/humane-image-gallery - Various dumped Humane image assets
